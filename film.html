FilmDB

Live Demo Repository

FilmDB is a movie database built using React where users can learn more about their favorite movies.

The web app features a home page that showcases trending movies and a search function to find additional films.

React Router is used to create this single-page app. Each movie description page is linked to, and can be accessed from, a unique URL.

FilmDB retrieves all movie information using the tmDB API.
