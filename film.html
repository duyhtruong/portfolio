FilmDB

Live Demo Repository

FilmDB was created with the intent of learning and practicing a modern JavaScript Library. FilmDB is a movie database web app created using React where users can learn about their favorite movies.

The web app fetches movie information from the TMDB API and returns a landing page filled with trending movies.

React saves user's search keyword using React States which is then passed as part of an endpoint to the TMDB API to return a list of movies that match.

The web app is routed using React Router. Users are able to use the browser's history including back and forward to visit different parts of the web page. Users may also visit unique URL's to learn more about their favorite movies.
