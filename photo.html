Photo Portfolio

Live Demo

Repository

My dad bought me my first camera when I was a freshman in high school. It was a Canon point and shoot. The early days of my new hobby consisted of taking (now embarassing) pictures of my friends at school, and since I couldn't drive at the time, many random pictures of my home.

Throughout the years, I've tried showcasing my photography on different social media platforms. Unfortunately, over time these platforms lose users, and I'm forced to move onto the next big thing.

Buying a domain and webpage for my photography from services like Square Space has always been an idea I've had in the back of my mind, but I never acted on it.

Creating this Photo Portfolio has been one of my motivators for learning web development.

Photo Portfolio is a responsive web page created to showcase my photography. It is structured using HTML, styled with CSS and Bootstrap, and has a filtering function incorporated using jQuery.

It features select photos in a large, minimal header utilizing Bootstrap's Carousel slideshow component.

Below the header, photos are organized using Bootstrap's responsive grid system. Custom CSS is added to each photo to keep them square. A filter feature was implemented using jQuery, displaying photos based on category.
