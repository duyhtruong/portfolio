Restaurant Roulette

Live Demo Repository

Restaurant Roulette is a web app created for indecisive eaters. The user chooses price-point, distance, and number of results, and Restaurant roulette returns a list of random restaurants nearby based on the user's criteria.

Geolocation API is used to get the user's current location. Google Maps API and Google Places Library are used to search for nearby restaurants.

Note* Location services must be on
