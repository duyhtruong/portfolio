Golden Hour

Live Demo Repository

In photography, 'Golden Hour' is the period of time just after the sun rises and sets over the horizon. Daylight during this time is diffused and even, with reduced shadows, and radiating with warm hues.

This minimal web app was created as a tool to help photographers take advantage of this lighting.

Users input a location in the form of address, city, or place of interest, and Golden Hour returns the time the sun rises and sets over the horizon. The web app features an animated gradient background from CSS Gradient Animator.

Google Maps API is used to geolocate the user's input location. The longitude and latitude is then used as part of the Sunrise Sunset API endpoint to retrieve the time the sun rises and sets in UTC. The time in UTC is then converted to the user location's local time using the Google Time Zone API.
